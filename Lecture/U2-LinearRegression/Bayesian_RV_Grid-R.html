just a Bayesian model of a RV (height), not a regression model¶

$h_i \sim N(\mu, \sigma)$

$\mu \sim N(\mu_\mu, \sigma_\mu) $

$\sigma \sim U(\sigma_L, \sigma_H) $

$h_i$ are iid and $\mu$ and $\sigma$ are independent

Grid approximation of the posterior¶

In [7]:
# parameter grids

ngrid = 200
mu.list <- seq( from = mean(df2$height)-sd(df2$height), to = mean(df2$height)+sd(df2$height), length.out = ngrid )
sigma.list <- seq( from = sd(df2$height)-1 , to = sd(df2$height)+1 , length.out = ngrid )


post <- expand.grid( mu = mu.list , sigma = sigma.list )   # (ngrid*ngrid) x 2

# log-likelihood of data given each combination of parameters
post$LL <- sapply( 1:nrow(post) , 
    function(i) sum( dnorm( df2$height , post$mu[i] , post$sigma[i] , log = TRUE ) ) 
                 )
                  
post$prod <- post$LL + dnorm( post$mu , mu_mean, mu_sd, log = TRUE ) + dunif( post$sigma , sigma_L, sigma_H, log = TRUE )

# RELATIVE posterior proba when scaled by max post prod, to bypass rounding to zero, 
post$prob <- exp( post$prod - max(post$prod) )
